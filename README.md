# AI Resume Analyzer

An AI-powered web app that analyzes resumes and provides feedback, role matching, and improvement suggestions.

## 🚀 Features
- Resume parsing
- AI-based feedback
- Role matching with confidence score
- Suggestions for improvement

## 🛠 Tech Stack
Frontend: React (Vite)  
Backend: Node.js, Express  
AI: OpenAI / LLM API  
Database: MongoDB (if used)

## 📦 Project Structure
frontend/
backend/

## ⚙️ Setup Instructions

### 1. Clone repo
git clone https://github.com/your-username/ai-resume-analyzer.git

### 2. Setup Backend
cd backend
npm install

### 3. Setup Frontend
cd ../frontend
npm install

### 4. Environment Variables
Create a `.env` file in backend:

(see `.env.example`)

### 5. Run App
Backend:
npm run dev

Frontend:
npm run dev
